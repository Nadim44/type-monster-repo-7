1. What is the difference between localStorage and sessionStorage?

Answer: Data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends.

2. What is the difference between globalScope and blockScope?

Answer: A globalScope variable means that the variable defined global will be accessible from outside the block and a blockscope variable means that the variable defined within a block will not be accessible from outside the block.

3. What is the JavaScript Event loop?

Answer: JavaScript has a runtime model based on an event loop, which is responsible for executing the code, collecting and processing events, and executing queued sub-tasks

4. How to get undefined?

Answer: 8 ways to get undefine:: 1. variable that is not initialized will give undefined. 2. function with no return 3. parameter that is not passed will be undefined. 4. if return has nothing on the right side will return undefined. 5. property that doesn't exists on an object will give you undefined. 6. accessing array elements outside of the index range. 7. deleting an element inside an array. 8. set a value directly to undefined.
